# 🧮 Matrix Operation Tool

A lightweight and efficient utility designed to perform various matrix operations seamlessly. Whether you are dealing with basic arithmetic or complex linear algebra computations, this tool provides a simple interface to handle your matrix manipulations.

## ✨ Features

* **Basic Arithmetic**: Addition, subtraction, and multiplication of matrices.
* **Scalar Operations**: Multiply or divide a matrix by a scalar value.
* **Advanced Operations**:
* Transposition ($A^T$)
* Determinant calculation ($|A|$)
* Inverse of a matrix ($A^{-1}$)
* Trace of a matrix


* **Utility Functions**: Check for symmetric, identity, or zero matrices.
* **Easy Input/Output**: Supports reading matrices from files or command-line input.
